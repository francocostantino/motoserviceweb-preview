# motoserviceweb-preview


Fullstack web application for managing motorcycle services, sales and inventory.

##  Technologies Used

- Node.js
- Express
- Sequelize
- MySQL
- HTML, CSS, JavaScript

##  Features

- User registration and login
- Product management panel
- Shopping cart system
- REST API architecture (MVC)
